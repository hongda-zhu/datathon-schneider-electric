feat: Implementación completa de deliverables del datathon Schneider Electric

Cumplimiento de los 4 deliverables requeridos:

1. Resumen de rendimiento del modelo
   - Métricas: F1 Score, AUC, Precision, Recall con tooltips explicativos
   - Distribución de predicciones y tasa de aciertos
   - Gráficos de distribución de probabilidad por buckets

2. Propuesta de técnicas de explicabilidad
   - SHAP global: summary plot con importancia de features
   - SHAP local: waterfall plot por oportunidad individual
   - Simulador What-If: análisis contrafactual interactivo
   - Guías de interpretación para cada técnica

3. Insights sobre predicciones Win/Loss
   - Explicación global de patrones del modelo
   - Análisis caso por caso con drivers positivos/negativos
   - Traducción de features técnicos a lenguaje de negocio
   - Resumen textual de drivers SHAP (top 3 positivos/negativos)

4. Interpretación automática de outputs SHAP
   - Generación de insights de negocio desde valores SHAP
   - Recomendaciones accionables por oportunidad
   - Resumen de patrones complejos en lenguaje comprensible

Características adicionales:
- Dashboard Streamlit con 3 páginas: Global Insights, Case Explorer, What-If Simulator
- Tooltips explicativos en todas las métricas
- Traducción completa de features a lenguaje de negocio
- Pipeline completo de entrenamiento y generación de insights
- Scripts de ejecución y documentación
